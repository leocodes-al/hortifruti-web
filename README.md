## 🚀 Sobre o projeto

O **HortiFruti Web** é um projeto front-end desenvolvido com o objetivo de simular uma aplicação real de e-commerce de frutas, verduras e legumes.

O projeto vai além da interface visual, contando com funcionalidades de carrinho de compras, persistência de dados e finalização de pedidos via WhatsApp, proporcionando uma experiência próxima de um sistema real.

## ✨ Funcionalidades

-  Navegação entre seções (Início, Produtos, Contato)
-  Listagem de produtos em layout de cards
-  Destaque de produtos em promoção
-  Carrinho de compras funcional
-  Adição e remoção de produtos
-  Controle de quantidade de itens
-  Persistência de dados com localStorage
-  Finalização de pedido via WhatsApp
-  Feedback visual ao adicionar produtos (toast)
-  Layout responsivo (desktop, notebook e mobile)
